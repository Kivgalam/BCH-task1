# BCH-task1
- <!DOCTYPE html>: Declares the document type and version of HTML.
- <html>: The root element of the HTML document.
- <head>: Contains metadata about the document, such as the title, charset, and links to stylesheets or scripts.
- <title>Hello World</title>: Sets the title of the page, which appears in the browser's title bar and in search engine results.
- <body>: Contains the content of the HTML document.
- <h1>Hello, World!</h1>: A heading element that displays the text "Hello, World!" in a large font size.
